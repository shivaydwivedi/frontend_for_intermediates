# Learning Path

This repository is the intermediate frontend step in a larger path:

```text
Beginner frontend -> Intermediate frontend -> Beginner backend -> Full-stack development
```

The learner entering this repository should already know HTML, CSS, responsive design, JavaScript fundamentals, DOM manipulation, forms, localStorage, Fetch API basics, async/await, public API integration, accessible interface basics, Git, and GitHub.

## What Changes At The Intermediate Stage

The projects here emphasize:

- Application architecture.
- Complex state.
- Accessible custom interactions.
- Async resilience.
- Testing strategy.
- Tooling decisions.
- TypeScript.
- React.
- Honest security and deployment limitations.

The goal is not to make projects look bigger. The goal is to make the reasoning more mature.

## Progression Summary

- Projects 01-03 deepen browser fundamentals.
- Projects 04-07 add richer state, data, testing, and selected libraries.
- Project 08 introduces TypeScript as the primary implementation language.
- Projects 09-10 introduce React and larger application architecture.

After this repository, the next stage should be beginner backend development so learners can understand real persistence, authentication, authorization, and server-side validation.
